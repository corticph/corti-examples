# Corti SDK Examples

This repository contains example applications for the Corti SDK, organized by language. Each language has its own directory and is a separate, self-contained project.

## Languages

| Language    | Directory   | Description                    |
|------------|-------------|--------------------------------|
| TypeScript | [typescript/](typescript/) | Next.js app with SDK examples (documents, facts, interactions, stream, etc.) and API route examples. |
| C#         | [csharp/](csharp/)        | Placeholder for C# example applications. |

Additional languages can be added as new top-level directories.

## Getting started

Open the directory for the language you want to use and follow that project’s README (e.g. [typescript/README.md](typescript/README.md)).
